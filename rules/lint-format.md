## Linting & Formatting

- Use `biome` as linter & formatter
- Lint: `bun run lint`
- Format: `bun run format`
- Check: `bun run check`

## TypeCheck

-`tsc --noEmit`

## Package Management

- Use `bun` as default package manager
- Install: `bun add [package-name]`
- Dev: `bun add -D [package-name]`
